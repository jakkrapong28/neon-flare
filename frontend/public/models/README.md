# 3D Avatar Model

Please place your 3D avatar file here.

**Filename:** `avatar.glb`
**Path:** `frontend/public/models/avatar.glb`

The `AvatarWidget` is configured to load this file automatically.
- If it contains an animation named "Idle", it will play it.
- If not, it will use the first available animation.
- If no file is present, a placeholder box will be shown.
